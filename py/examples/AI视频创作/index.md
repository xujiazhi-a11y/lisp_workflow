# AI 视频创作工作流 · 模块说明

## 文件职责

| 文件 | 行数 | 职责 |
|---|---|---|
| `主入口.lisp` | 22 | 工作流总入口，按顺序引入所有子模块 |
| `输入读取.lisp` | 48 | 读取 `用户输入数据`，校验必填字段，初始化全局变量（参考图、故事描述、段数、角色描述、单段时长、目标时长） |
| `辅助函数.lisp` | 58 | 底层工具函数：格式化图片提示词、生成图片、生成首尾帧候选、显示候选帧、生成分镜描述、生成/优化视频描述、生成视频 |
| `交互选帧.lisp` | 33 | 交互式选帧：生成候选图 → 用户选择「使用这组帧」或「重新生成」（递归循环） |
| `交互优化描述.lisp` | 33 | 交互式优化视频描述：生成初始描述 → 用户选择「使用」/「重新生成」或输入修改意见（递归循环） |
| `交互生成视频.lisp` | 46 | 交互式视频生成：生成视频 → 用户选择「使用当前版本」/「重新生成」/「选历史版本」（递归循环，保留历史） |
| `单段处理.lisp` | 38 | 单段处理主函数：串联分镜生成 → 交互选帧 → 交互优化描述 → 交互生成视频，返回该段结果字典 |
| `主循环.lisp` | 17 | 递归逐段处理循环：从第 1 段到第 N 段依次调用 `处理单段`，收集所有段结果 |
| `拼接成片.lisp` | 38 | 收集所有段视频路径，判断是否需要拼接，输出最终成片路径并打印完成信息 |

## 对外接口

- **入口**：`主入口.lisp`（被前端点击执行）
- **输入**：`用户输入数据` 字典（由 `workflow_server.py` 注入），必须包含 `"参考图"`、`"故事"`、`"段数"`
- **输出**：最终视频路径（字符串）
- **副作用**：打印进度信息、调用 AI 图像/文本/视频 API、触发前端交互卡片

## 模块依赖图

```
主入口.lisp
  ├── 输入读取.lisp      （定义全局变量）
  ├── 辅助函数.lisp      （依赖输入读取的全局变量）
  ├── 交互选帧.lisp      （依赖辅助函数）
  ├── 交互优化描述.lisp  （依赖辅助函数）
  ├── 交互生成视频.lisp  （依赖辅助函数）
  ├── 单段处理.lisp      （依赖交互选帧/优化描述/生成视频）
  ├── 主循环.lisp        （依赖单段处理）
  └── 拼接成片.lisp      （依赖主循环的结果）
```

## 编码规范

- 使用中文方括号 `【】` 代替英文括号
- 函数和变量使用中文命名
- 单个文件不超过 50 行
- 嵌套超过 3 层时，闭合括号后加注释标明关闭的块
- 递归实现循环（`如果` + 自调用）
